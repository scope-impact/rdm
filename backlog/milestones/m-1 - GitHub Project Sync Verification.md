## Description

Bidirectional sync between Backlog.md and GitHub. Push tasks/milestones as GitHub Issues and Projects for team visibility. Pull PRs and reviews back into DuckDB for IEC 62304 traceability and change history documentation.

## Features

- RDM-001: Sync Backlog.md tasks to GitHub Issues
- RDM-002: Sync Backlog.md milestones to GitHub Projects and Milestones
- RDM-003: Sync GitHub PRs and Issues back to DuckDB for traceability
