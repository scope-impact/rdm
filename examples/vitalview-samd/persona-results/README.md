# persona-results/

Formative usability evidence (`*-persona.json`) is **generated by the
`usability-persona` skill** (`.claude/skills/usability-persona/`), not authored
by hand. Each run drives the VitalView dashboard as a persona (see `../personas/`)
and writes one record here, which `rdm story persona` ingests.

To populate this directory: run the dashboard, then invoke the skill for each
persona spec (it calls `scripts/write_evidence.py` to emit the JSON). Then:

```bash
rdm story persona \
  --vv-plan ../docs/verification_and_validation_plan.md \
  --persona-results .
```

These are **formative** evidence only — never summative IEC 62366 validation.
